# Story 4.6: 发送邮件提醒并保留失败状态

Status: done

## Story

As a 个人用户,
I want 启用邮件提醒后在满足提醒条件时收到邮件,
so that 即使没有打开网页也能被提醒。

## Acceptance Criteria

1. Given 用户已启用邮件提醒，并已配置有效 SMTP 和收件邮箱, when `ReminderRunner` 判定当天满足提醒条件, then 系统通过 Nodemailer 发送邮件提醒。
2. 邮件内容使用中文。
3. 当天同类型同渠道邮件提醒不重复发送。
4. 发送成功时记录提醒事件状态。
5. 发送失败时记录失败状态和中文可读错误摘要。
6. 页面可以查看最近邮件提醒发送状态。
7. SMTP 未配置完整时不尝试发送邮件，并显示配置依赖提示。
