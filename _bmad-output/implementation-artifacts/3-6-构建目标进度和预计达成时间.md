# Story 3.6: 构建目标进度和预计达成时间

Status: done

## Story

As a 个人用户,
I want 看到当前值、目标值、差距和预计达成时间,
so that 我知道离目标还有多远，以及当前趋势是否有效。

## Acceptance Criteria

1. Given 用户已设置健康目标或跑步目标, when 首页或目标页读取目标进度, then 系统显示当前值、目标值、剩余差距和目标状态。
2. 目标状态包含未设置、进行中、已达成、落后。
3. 系统使用最近记录估算预计达成时间。
4. 未读取到用户配置的阈值时，系统使用默认最低条件：至少 7 天、至少 3 条有效记录。
5. 数据不足、低于最低条件或趋势反向时显示“无法可靠估算”或等价中文原因。
6. 进度条同时提供中文数值说明，不能只靠颜色表达。
7. 编辑或删除历史记录后，目标进度读取最新源事实重新计算。
