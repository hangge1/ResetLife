# Story 3.3: 设置和展示跑步目标

Status: done

## Story

As a 个人用户,
I want 在“目标”页面设置每周跑步目标,
so that 我可以按次数和跑量管理运动计划。

## Acceptance Criteria

1. Given 用户在受信浏览器进入“目标”页面, when 用户填写每周跑步次数和每周跑量并保存, then 系统保存跑步目标。
2. 表单显示中文标签和单位。
3. 跑步次数必须为有效的大于 0 的整数。
4. 每周跑量必须为有效的大于 0 的数字。
5. 非法数值在字段旁显示中文错误并阻止保存。
6. 页面展示当前跑步目标摘要。
