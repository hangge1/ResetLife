# Story 4.7: 管理访问密码和受信设备

Status: done

## Story

As a 个人用户,
I want 在设置中修改访问密码并移除不再信任的设备,
so that 我可以维护当前轻量访问保护状态。

## Acceptance Criteria

1. Given 用户在受信浏览器进入设置页面, when 用户修改访问密码, then 系统要求输入当前访问密码和新访问密码。
2. 当前访问密码错误时显示中文错误。
3. 新访问密码保存时服务端只保存哈希。
4. 修改成功后显示中文成功提示。
5. 受信设备列表显示设备或浏览器名称、创建时间、最近访问时间和移除操作。
6. 移除受信设备前显示中文确认提示。
7. 移除当前设备后当前浏览器需要重新验证。
8. 未受信浏览器不能访问访问保护设置。
