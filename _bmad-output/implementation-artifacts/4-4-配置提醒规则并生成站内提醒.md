# Story 4.4: 配置提醒规则并生成站内提醒

Status: done

## Story

As a 个人用户,
I want 设置每日提醒时间、站内提醒开关和邮件提醒开关,
so that 我在忘记记录或运动时能看到非打扰式提醒。

## Acceptance Criteria

1. Given 用户在设置页面配置提醒规则, when 用户设置每日提醒时间，并配置站内提醒开关和邮件提醒开关, then 系统保存提醒规则。
2. 页面清晰显示邮件提醒依赖 SMTP 配置和收件邮箱。
3. `ReminderRunner` 到达提醒窗口后读取提醒规则和当天记录状态。
4. 当用户在提醒时间前未完成指定动作时，系统生成站内提醒。
5. 今日状态已完成时不生成重复提醒。
6. 站内提醒按 `localDate + reminderType + channel` 保持幂等。
7. 首页今日状态条读取并展示最新提醒状态。
8. 页面只展示提醒状态，不重新判定发送资格。
9. 站内提醒以页面内条目展示，不使用阻塞弹窗。
