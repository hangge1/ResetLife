# Story 2.4: 浏览和筛选历史记录

Status: done

## Story

As a 个人用户,
I want 查看和筛选历史健康与跑步记录,
so that 我可以回顾自己的记录而不是翻表格。

## Acceptance Criteria

1. Given 系统中已有健康记录和跑步记录, when 用户打开“历史”页面, then 系统按日期展示健康记录和跑步记录。
2. 用户可以按全部、健康、跑步筛选。
3. 用户可以按最近 7 天、最近 30 天或自定义日期范围筛选。
4. 历史页面显示中文单位和记录摘要。
5. 移动端可读，不依赖宽表格。
6. 空数据时显示中文空状态。
7. 未受信浏览器不能访问历史页面。
