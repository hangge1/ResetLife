# Story 2.5: 编辑和删除历史记录

Status: done

## Story

As a 个人用户,
I want 修改或删除错误记录,
so that 历史数据和后续计算保持准确。

## Acceptance Criteria

1. Given 用户在历史页面查看一条记录, when 用户选择编辑, then 系统打开可编辑表单并保留原始值。
2. 保存后记录更新。
3. 修改健康记录时仍遵守同日唯一规则。
4. 修改或删除后相关首页、趋势、目标进度读取最新源事实。
5. When 用户选择删除记录, then 系统显示中文确认提示。
6. 用户确认后才删除记录。
7. 删除失败时显示中文错误并不移除当前页面记录。
8. 未受信浏览器不能访问编辑页面或提交编辑/删除入口。
