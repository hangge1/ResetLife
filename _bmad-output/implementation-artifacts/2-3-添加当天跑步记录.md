# Story 2.3: 添加当天跑步记录

Status: done

## Story

As a 个人用户,
I want 录入一次跑步活动,
so that 一天内多次跑步都能被单独记录。

## Acceptance Criteria

1. Given 用户在受信浏览器进入“记录”页面, when 页面加载, then 系统显示中文跑步记录表单。
2. 表单字段包含公里数、运动时长、配速、平均心率、平均步幅、步频，并显示中文标签和单位。
3. 保存时系统创建一条当天跑步记录。
4. 同一天可以创建多条跑步记录，彼此使用不同 UUID。
5. 如果用户填写公里数和运动时长但未填写配速，系统按距离和时长计算配速。
6. 如果用户手动填写配速，系统保存手动配速。
7. 非法数值在字段旁显示中文错误，并保留用户已输入内容。
8. 保存后记录仍归属到用户本地日期。
9. 未受信浏览器不能访问记录页面或提交保存入口。
