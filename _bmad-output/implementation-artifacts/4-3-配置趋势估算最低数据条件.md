# Story 4.3: 配置趋势估算最低数据条件

Status: done

## Story

As a 个人用户,
I want 设置预计达成时间所需的最低数据条件,
so that 估算结果不会基于太少的数据误导我。

## Acceptance Criteria

1. Given 用户在设置页面打开趋势估算配置, when 用户填写最低统计天数和最低有效记录数并保存, then 系统保存趋势估算阈值。
2. 最低统计天数不得低于 7 天。
3. 最低有效记录数不得低于 3 条。
4. 低于系统下限时阻止保存并显示中文说明。
5. 首页目标进度和预计达成时间使用最新阈值。
6. 未配置时系统使用默认值：7 天和 3 条有效记录。
