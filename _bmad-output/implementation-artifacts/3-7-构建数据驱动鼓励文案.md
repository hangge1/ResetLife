# Story 3.7: 构建数据驱动鼓励文案

Status: done

## Story

As a 个人用户,
I want 首页根据近期记录和目标进度给出简短鼓励,
so that 我能获得具体、克制、可行动的反馈。

## Acceptance Criteria

1. Given 系统已有近期记录、今日状态和目标进度, when 用户打开首页, then 系统显示一条中文鼓励文案。
2. 文案与当前数据状态一致。
3. 连续记录、接近目标、已达成、数据不足、趋势反向等状态有不同反馈。
4. 文案不使用羞辱性、医疗化或夸大确定性的表达。
5. 文案保持简短，适合首页卡片展示。
6. 无目标或无记录时显示中文引导文案。
