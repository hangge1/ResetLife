# Story 3.2: 设置和展示健康目标

Status: done

## Story

As a 个人用户,
I want 在“目标”页面设置健康目标,
so that 我可以明确当前身体指标想达到的方向。

## Acceptance Criteria

1. Given 用户在受信浏览器进入“目标”页面, when 用户填写目标体重，以及可选的目标腰围、目标臀围、目标体脂率并保存, then 系统保存健康目标。
2. 表单显示中文标签和单位。
3. 非法数值在字段旁显示中文错误并阻止保存。
4. 未填写的可选目标不会被当成 0 参与计算。
5. 保存成功后页面显示中文成功提示。
6. 页面展示当前健康目标摘要。
