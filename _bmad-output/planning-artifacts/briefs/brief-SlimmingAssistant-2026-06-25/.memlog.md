- (event) Create product brief from doc/requirement.md; current source describes a personal web-based health assistant for workout/health tracking, trend feedback, and reminders.
- (decision) MVP scope confirmed: single personal user, manual data entry only, Web app first, reminders limited to in-app and email; SMS/WeChat/QQ deferred.
- (decision) Daily health data is one record per day; repeated submission overwrites the same day's record. MVP includes configurable goals such as weekly run count and weekly mileage. Dashboard treats health and exercise as equally important, shows distance to goals, estimates time to goal based on trend, and includes encouragement copy.
